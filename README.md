# vendas_xbox-dio
Descrição do Projeto

Projeto de análise de dados de assinaturas Xbox utilizando Excel. O objetivo é entender o comportamento dos assinantes, identificar os planos mais usados, calcular receita e avaliar cancelamentos, tudo a partir de uma planilha estruturada e gráficos.

Descrição dos Dados Utilizados

A base contém informações fictícias de assinantes Xbox, incluindo:

ID do usuário

Nome

Tipo de plano (Game Pass, Ultimate, Core…)

Valor pago

Data de início

Status da assinatura

Tempo ativo em meses

Esses dados permitem calcular métricas e criar visualizações.
